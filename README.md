# Hello, Static Site Generator
Static Site Generator that converts markdown into HTML pages.

> This project is being developed as part of [boot.dev](https://boot.dev)'s curriculum.

## Development Roadmap
- [x] Text to Text Nodes
- [ ] Split Blocks
- [ ] Block Types
- [ ] Block to HTML

## Testing
Run unit tests with:
```bash
./test.sh
```
