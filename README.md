# Hello, SSG!
This is a static site generator (SSG) project that converts markdown
files into HTML pages.

> This project is a work in progress, not functional yet.

## Development Tasks
- [x] Text to Text Nodes
- [ ] Split Blocks
- [ ] Block Types
- [ ] Block to HTML

## Testing
To unit test the project, you can run the test.sh script:
```bash
./test.sh
```
